# Telegram Shop Bot

Простой бот-магазин для Telegram.

## Как запустить

1. Установи зависимости: `pip install aiogram python-dotenv`
2. Создай файл `.env` с токеном бота
3. Запусти: `python main.py`

## Файлы проекта

- `main.py` - основной код бота
- `config.py` - настройки
- `.env` - токен бота
- `requirements.txt` - список библиотек
